Tutorial Básico de JavaScript

JavaScript es un lenguaje de programación orientado a objetos que se ejecuta en el navegador del cliente. Se utiliza principalmente para crear interactividad en sitios web, permitiendo a los desarrolladores implementar funcionalidades dinámicas y responder a eventos del usuario.

1. INTRODUCCIÓN. Configuración Inicial

Incluir Javascript en HTML

Para comenzar a utilizar JavaScript, crea un archivo llamado script.js y luego inclúyelo en tu documento HTML utilizando la etiqueta <script>. Es recomendable colocar el enlace al archivo JavaScript justo antes del cierre de la etiqueta </body> para asegurar que el contenido de la página se haya cargado antes de ejecutar el script.

Si decides incluir el script en la parte inferior de tu documento HTML, debería verse así:


<!DOCTYPE html>
<html>
<body>

<h1>Titulo</h1>

<p>Parrafo</p>
<script src="script.js"></script>

</body>
</html>

Ejemplo de Código JavaScript en script.js


    // Este código imprime "HOLA MUNDO" en la página
    document.write("HOLA MUNDO");

2. VARIABLES

Variables

Las variables son contenedores para almacenar datos. Pueden definirse con let, o const si es un valor que no va a cambiar.

5. DOM. MODIFICAR ELEMENTOS

javascript Modificar Elementos del DOM

El DOM (Document Object Model) permite acceder y modificar el contenido de los elementos HTML. Puedes cambiar el texto de un elemento utilizando la propiedad textContent.

HTML


<p id="miTexto">Este es un texto que se puede cambiar.</p>
<button onclick="modificarTexto()">Cambiar Texto</button>

JavaScript


function modificarTexto() {
    document.getElementById("miTexto").textContent = "¡Texto cambiado!";
}

Este es un texto que se puede cambiar.

5. DOM. MODIFICAR ESTILOS

javascript Modificar Estilos del DOM

Puedes cambiar el estilo de un elemento mediante style. Ejemplo: cambiar color, ocultar o mostrar un elemento.

7. CONDICIONALES

Condicionales

Los condicionales permiten ejecutar código en función de condiciones. Usa if, else if, y else. 


            function ejemploCondicionales() {
                let edad = 18; // Puedes cambiar este valor para probar diferentes resultados
                let mensaje = "";
                
                if (edad >= 18) {
                    mensaje = "Eres mayor de edad.";
                } else {
                    mensaje = "Eres menor de edad.";
                }
                
                // Mostrar el mensaje en el elemento con id "resultadoCondicionales"
                document.getElementById("resultadoCondicionales").textContent = mensaje;
            }

9. ARRAYS

Los arrays son listas de valores. Puedes agregar, eliminar y ordenar elementos en un array.

Arrays 

            function ejemploArrays() {
                let numeros = [3, 1, 2];
                // Eliminar el último elemento y mostrar el array
                numeros.pop();
                document.getElementById("resultadoArrays").textContent += "Eliminar: " + numeros + " ";
                // Agregar elemento y mostrar el array
                numeros.push(4);
                document.getElementById("resultadoArrays").textContent += "Agregar: " + numeros + " ";
                //Obtener la longitud del array
                let longitud = numeros.lenght;
                document.getElementById("resultadoArrays").textContent += "Longitud: " + longitud;
                // Ordenar el array y mostrarlo
                numeros.sort();
                document.getElementById("resultadoArrays").textContent +="Ordenar: " + numeros + " ";
                // Obtener el elemento de la posicion que elijas, el primero es el 0
                let primero = numeros[0];
                let tercero = numeros[2];
                document.getElementById("resultadoArrays").textContent += "Primero " + primero + "Tercero " + tercero;
            
            }

Bucle For

Los bucles for permiten repetir acciones un número específico de veces.

Bucle For 

            function ejemploFor() {
                let resultado = "";
                for (let i = 0; i < 5; i++) {
                    resultado += "Número: " + i;
                }
                document.getElementById("resultadoFor").textContent += "Numeros " + resultado;
            }

            function ejemploForArray() {
                let lista = ["Ana", "Maria", "Juan"];
                let resultado = "";
                for (let i = 0; i < lista.lenght; i++) {
                    resultado += "Nombre: " + lista[i];
                }
                document.getElementById("resultadoFor").textContent += " Lista " + resultado;
            }
